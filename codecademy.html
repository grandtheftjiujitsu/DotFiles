General Information

  • HTML files are opened using a browser which then renders the file.
  • The "head" section is printed as a header(different from a heading).
  • The "title" is NOT printed. It is like metadata for the page.

  • Rich Text

    • This is done under a "style" section within the opening text tag ("p" or "h1" for example)

    • Size

      • style="font-size:XXpx"
      • *for relative sizing use "em" instead of "px"
        • Do this by dividing content size (XXpx) by default (most) browser fonts size (16px)
        • 16px (font) / 16px (browser default) = 1.000em

    • Color

      • Similarly, use "color:color-name"
      • Violet Example
      • Or use a HEX color (#00ff00) insead of a color's name
      • A good list of HEX colors can be found at www.colorhexa.com

    • Font Families

      • The style feature used here is "font-family:Family-Name"
      • Be sure to capitalize the first letter of the Family-Name (e.g.: Verdana, Impact, Arial, etc...)
      • Examples of Common Fonts:
        • Default (browser dependent)
        • Arial
        • Georgia
        • Verdana
        • Impact
        • Times New Roman
        • Lucida
        • Clean
        • Lucida

    • Background Color

      • Simply use "background-color:color-name" inside the "style" section of the first tag
    • Text Alignment
      • Another "style" feature using "text-align: alignment"
      • Center Align
      • Right Align

    • Nesting Styles

      • There should only be ONE "style" section per tag pair
      • Inside the quoted section use a "; " to separate style features (style="color: green; font-size: 12px")

Hyperlinks

  • Start with an "a" tag
  • Within the first "a" use --> href="www.whatever.com"
  • Also within the first "a" you can also add --> title="title" -- which prints as a tooltip when hovered over.
  • Between the "a" and "/a" tags is where the printed / displayed text goes
  • Example: Google Hyperlink

Lists

  • Ordered / Numbered Lists
    1. Start with "ol" tag
    2. Then use "li" tag for "list item"
  • Unordered / Bulleted Lists
    • Start with "ul" tag
    • Use "li" tag for "list item"
